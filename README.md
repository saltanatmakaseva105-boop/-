## Описание

Training Planner — это GUI‑приложение для планирования тренировок. Позволяет:
* Добавлять тренировки с указанием даты, типа и длительности.
* Фильтровать записи по типу тренировки и дате.
* Сохранять и загружать данные в формате JSON.
* Проверять корректность ввода данных.

## Требования

* Python 3.6 или выше.
* Стандартные библиотеки: `tkinter`, `json`, `datetime`.
